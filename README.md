# tutor
Tutorial hosting web application with django 

# requirement
- Python3.7
- Django2.2.7
- Docker 19.03.2

# Project Setup 

1. Building a docker image 
```bash
$ docker-compose build
```

2. Install Django library
```bash
docker-compose run --rm app pipenv install
```

3. start docker container
```bash
$ docker-compose up 
```
